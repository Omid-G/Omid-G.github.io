•  Frame •  Seven Segment Display •  Touch Capacitive Button •  Microcontroller with onboard ESP8266 Wifi Chip •  3D printer •  Micro usb cable •  Soldering Iron & solder •  Wires & wire strippers
To hook up the circuit you will need to connect the seven segment display and the touch capacitive button to your microcontroller as shown in the schematic. The exact pin outs will depend on the specific microcontroller being used.
This project makes use of the Arduino Youtube API written by witnessmenow on Github and the TM1637 Library to drive the seven segment display. You can find instructions on how to use the Youtube API here, including how to set up a Google Apps API key and find your channel ID. I’ve written a program that uses the Youtube APi, TM167 Library and incorporates a touch capacitive button which you can download here. However, you will need to replace your network ID, password, API key and channel ID with that of your own.
To    download    the    source    code    and    printable    files    visit    the    Github    and Thingiverse links below. For additional information email me at OmidGhayouri@outlook.com

The Live Youtube Subscriber Counter uses a micro controller with a ESP8266 WiFi chip and a seven segment display to show any Youtube channel’s subscriber count in real time. The project requires a few materials, basic soldering skills, and access to a 3D printer. All of the source code and 3D files are available to download at the bottom of this page.
